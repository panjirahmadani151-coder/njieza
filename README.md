# njieza

A modern minimalist clothing e-commerce prototype inspired by Uniqlo.

## Files
- `index.html` — Homepage with featured collections, product highlights, and O2O promo.
- `product.html` — Product detail page with color swatches, size assistant, reviews, and store pickup.
- `checkout.html` — One-page checkout flow with Click & Collect option.
- `styles.css` — Design system tokens and responsive layout.
- `script.js` — Cart state, product interactions, size recommendations, and checkout behavior.

## Run locally
Open `index.html` in a browser or use a local static server.

## Notes
This prototype uses a static, responsive structure with interactive cart and checkout simulation.
